# 安全策略

[English](SECURITY.md) | **简体中文**

## 报告漏洞

请 **不要** 将安全问题作为公开 GitHub Issue 提交。

请优先使用以下方式之一：

1. [GitHub 私密漏洞报告](https://github.com/happyfeetw/grok-cli/security/advisories/new)
   （若本仓库已开启），或
2. 通过维护者 [happyfeetw](https://github.com/happyfeetw) 主页上的联系方式私下沟通。

请尽量提供复现步骤、影响评估，以及你已有的修复建议。我们会在可行范围内尽快确认，
并协商披露方式。

## 范围

本仓库是开源 Grok Build 的 **社区 fork**。仅适用于 SpaceXAI / xAI 官方托管服务的
问题，请走上游渠道，而非本 fork。
