# Security Policy

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security reports.

Prefer one of the following:

1. [GitHub private vulnerability reporting](https://github.com/happyfeetw/grok-cli/security/advisories/new)
   (if enabled for this repository), or
2. Contact the repository maintainers privately via the email or channel listed
   on the maintainer profile for [happyfeetw](https://github.com/happyfeetw).

Include steps to reproduce, impact assessment, and any suggested fix if you
have one. We will acknowledge receipt as soon as practical and coordinate
disclosure.

## Scope

This repository is a **community fork** of open-source Grok Build. Reports
that only apply to official SpaceXAI / xAI hosted services should be sent
through upstream channels, not this fork.
